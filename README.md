#  Fahrenheit to Celsius Converter GUI

A fun and interactive **GUI-based temperature converter** built using **Python and Tkinter**.  
This application allows users to easily convert temperatures from Fahrenheit to Celsius through a simple and user-friendly interface.

## Project Overview

The Fahrenheit to Celsius Converter is a beginner-friendly Python GUI project that demonstrates:

- Building graphical user interfaces using Tkinter
- Handling user input
- Performing mathematical conversions
- Adding interactive buttons and error handling

## Features

-  Convert Fahrenheit temperature to Celsius instantly
-  Simple and interactive graphical interface
-  Real-time conversion
-  Input validation for incorrect values
-  Clear button to reset input
-  Built completely with Python

##  Technologies Used

- **Python**
- **Tkinter** (Python GUI Library)

## Conversion Formula

Celsius = (Fahrenheit - 32) × 5/9
